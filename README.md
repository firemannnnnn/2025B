# 2025B - 半导体外延层光学分析

基于光学反射谱的半导体外延层厚度与掺杂浓度分析项目。

## 项目概述

本项目针对半导体外延层的光学反射谱数据进行分析，包含三个核心问题：

- **问题1**：折射率模型建立与验证
- **问题2**：SiC外延层厚度确定算法与可靠性分析
- **问题3**：多光束干涉效应分析

## 问题说明

### 问题1：折射率模型

建立Si和SiC材料的折射率模型，考虑自由载流子吸收效应：

- Lorentz-Drude模型描述折射率随波数变化
- 分析不同掺杂浓度对折射率的影响
- 验证模型与实验数据的一致性

### 问题2：SiC外延层厚度确定

采用多种方法综合确定外延层厚度：

- 相邻极值法
- 线性回归法
- FFT法
- 差分进化曲线拟合法（双光束/多光束模型）
- Bootstrap不确定性评估

**优化改进**：
- 采用非偏振光模型（s+p偏振平均）
- 引入差分进化全局优化算法
- 添加多光束干涉模型支持
- 双光束与多光束对比分析

### 问题3：多光束干涉分析

分析多光束干涉效应对厚度测量的影响：

- 计算多光束显著性参数 β = |r₁₀·r₁₂|
- 判断双光束近似的适用条件
- 对比Si和SiC材料的多光束干涉效应差异

**判定标准**：
- β < 0.05：双光束近似成立
- 0.05 < β < 0.2：多光束干涉可测量
- β > 0.2：必须使用Fabry-Perot模型

## 核心算法

### 反射率模型

三层介质（空气/外延层/衬底）干涉模型：

- 双光束近似：仅考虑两次反射
- 多光束模型：Fabry-Perot干涉

### 参数拟合

采用差分进化算法进行全局优化，避免局部最优解。

### 可靠性评估

- MAE/RMSE/R²：拟合质量指标
- Bootstrap重采样：不确定性评估
- 双光束vs多光束对比：模型适用性判断

## 结论

1. SiC材料的多光束干涉效应极小（β≈0.001），双光束近似完全可靠
2. Si材料在高掺杂衬底情况下多光束干涉显著，必须使用多光束模型
3. 差分进化算法能有效避免局部最优，提高厚度测量精度
